รายละเอียดการลงทะเบียน
เงื่อนไขในการลงทะเบียน

1. ผู้สมัคร จะต้องกรอกข้อมูลรายละเอียดต่างๆ ตามจริงให้ครบถ้วน

2. หากตรวจพบว่าข้อมูลของผู้สมัครไม่เป็นความจริง จะไม่อนุมัติการลงทะเบียน

3. เมื่อลงทะเบียนสำเร็จแล้ว โปรดติดตามการประกาศผลในหน้าเว็บคณะกรรมการนักเรียน

4. โปรดจำรหัสผ่านที่ระบบให้กรอก เพื่อใช้ในการเข้าระบบอีกครั้งหลังประกาศผล
